# Summary - TypeBot

## General

### The Basics

- The purpose of TypeBot is to provide a polished front end for a web chatbot.
  It's best used along with n8n as a back end

- TypeBot has a native WordPress plugin

- TypeBot's license allows us to use it for anything without any restrictions,
  only with less support options. So there's no need to use the cloud version
  for clients, like with n8n

- For clients that want full control of their installation, I can create a
  seperate elast.io installation. For other clients, I can use my own
  installation, and keep scaling it up when needed

- The only thing that TypeBot lacks is user dashboards for the clients. So for
  clients that need that, WotNot and LandBot can be good options
